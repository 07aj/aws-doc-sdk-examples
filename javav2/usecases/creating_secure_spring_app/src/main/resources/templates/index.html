AWS Item Tracker

The AWS Item Tracker application is a sample application that uses multiple AWS Services and the Java V2 API. Collecting and working with items has never been easier! Simply perform these steps:

  1. Enter work items into the system by clicking the Add Items menu item. Fill in the form and then click the Create Item button.
  2. The AWS Item Tracker application stores the data by using the Amazon Relational Database Service (Amazon RDS).
  3. You can view all of your items by clicking the Get Items menu item. Next, click the Get Active Items button in the dialog.
  4. You can modify an Active Item by selecting an item in the table and then clicking the Get Single Item button. The item appears in the Modify Item section where you can modify the description or status.
  5. Modify the item and then click the Update Item button. Note that you cannot modify the ID value.
  6. You can archive any item by selecting the item and clicking the Archive Item button. Notice that the table is updated with only active items.
  7. You can display all archived items by clicking the Get Archived Items button. Note that you cannot modify an archived item.
  8. You can send an email recipient an email message with a report attachment by selecting the email recipient from the dialog and then clicking the Report button. Note only Active data can be sent in a report.
  9. The Amazon Simple Email Service is used to send an email with an Excel document to the selected email recipient.
